【PHP】正規表現メモ

 Posted on 2020/12/9

目的: CSSプロパティで指定された画像URLを取り出したい

background-image: url( https://xxxxx.png );


function getUrl($url)
{
  preg_match('/(https).*\.(png|jpg|jpeg)/i', $url, $match);
  return $match[0] ?? null;
}

$url = getUrl("background-image: url( https://xxxxx.png );");
echo $url;



$ php index.php 
"https://xxxxx.png"
